该项目为一个简化版的网上商城，包括完整的网上销售功能：
普通用户的浏览和购买功能、网站管理员后台管理功能。
前端采用了 **AngularJs+BootStrap** 搭建页面和视图控制，后台使用 **Spring+Mybatis+Redis**。
实现了用户登录(基于 LoginToken 的会话登录)、注册、个人信息修改、商品浏览及搜索、订单管理以及后
台管理系统等功能。
